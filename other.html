Clearfix

The clearfix class can be used to clear floated elements. Use this when you wish to give put the element on it's own line without other elements floating up around it.

/* The CSS generated by the Clearfix class */
.clearfix {
    *zoom: 1;
}
.clearfix:before, .clearfix:after {
    display: table;
    content: "";
}
.clearfix:after {
    clear: both;
}
