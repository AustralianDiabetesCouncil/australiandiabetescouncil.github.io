Purpose of This Document

This document is intended as an informational reference guide when creating content to be used on the Australian Diabetes Council website. Please take the time to read and understand how the various elements work, and how they can be used together to ensure a vital brand identity.

How to Use This Document

Use the navigation on the left to find the element you would like to use based on its primary category. Within each element you will find a short explanation of the elements purpose, an example of its usage and in most cases the code required to output that element on the website.
